Esta app permite la revision de las mascotas de un refugio de animales, permite ver las mascotas, distinguir su especie y filtrar busqueda por su especie y revisar que mascotas estan en necesidad de adopcion.

Para ejecutar este proyecto se debe entrar al proyecto, y en la terminal ejecutar usando npm run dev, y luego abrir la URL indicada.

Se declara uso de IA en el asistente github copilot.
